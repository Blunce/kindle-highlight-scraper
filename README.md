## Installation & Use
1. Install [Easy Install](http://peak.telecommunity.com/DevCenter/EasyInstall#installing-easy-install)
2. Run ``easy_install mechanize``
3. Run ``extract_kindle_notes.py``
4. Provide Amazon email & password when prompted
